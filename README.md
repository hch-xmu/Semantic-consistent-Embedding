# Semantic-consistent Embedding
This is the toy data example of semantic-consistent embedding (SCE).

# Requirements
~~~
python=3.8
pytorch=1.9.0
scikit-learn=0.24
~~~
This code was tested with Pytorch 1.9.0, CUDA 10.2 and Windows 10.  </br>

# Quick start
```shell
python sce.py
```
